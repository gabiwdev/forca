# Jogo da Forca

Mini projeto de um jogo da forca em Python, criado para reforçar conceitos básicos de lógica e estruturação de código.

## Funcionalidades
- O jogo escolhe uma palavra aleatória
- O jogador deve adivinhar a palavra chutando letras
- O jogador possui apenas 6 tentativas de erro
- Letras repetidas não consomem tentativa

## Aprendizado
- Uso de loops `while` e `for`
- Criação e uso de funções para organização do código
- Uso de listas, dicionários e conjuntos (`set`)
- Compreensão geral da lógica de programação
